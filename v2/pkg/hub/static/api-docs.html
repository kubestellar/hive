API Reference

Hive Hub REST API — all endpoints for the hub at hive.kubestellar.io

Public Authentication Dashboard Hosted Hives Access Control Admin Spoke Integration

Public Endpoints

GET /api/registry public
Returns all public hive instances with their status, agents, ACMM level, governor mode, and contributor counts. Stale offline entries with hosted duplicates are filtered.
GET /api/hub/leaderboard public
Cross-hive contributor leaderboard. Merges contributors across all public hives, deduplicates by GitHub username. Includes active status, current task, hive name.
GET /api/hub/stats public
Aggregate stats: total hives online, agents running, active contributors, issues, PRs.

Authentication

GET /login public
Redirects to GitHub OAuth authorization. Accepts ?redirect=URL to return to a specific page after login. Scope: read:user,repo.
GET /api/auth/callback public
GitHub OAuth callback. Exchanges code for token, encrypts and stores it, creates user profile, sets hive_hub_user cookie on .hive.kubestellar.io domain.
GET /api/auth/user public
Returns {"authenticated": true, "login": "...", "avatar_url": "..."} if the user has a valid cookie, or {"authenticated": false}.
POST /api/auth/logout login
Clears the authentication cookie and redirects to home.

Dashboard

GET /api/saas/my-hives login
Returns hives the authenticated user owns or has access to. Includes local hives (matched by heartbeat owner field) and hosted hives (from metadata). Also returns saas_quota, saas_used, is_admin.

Hosted Hive Management

POST /api/saas/hives login + quota
Create a new hosted hive instance on the OKE cluster. Body: {"org", "repos", "primary_repo", "project_name", "acmm_level", "auth_method": "pat"|"app", "github_token"|"app_id"+"installation_id"+"app_private_key"}. Validates quota, creates K8s namespace/deployment/service/ingress.
GET /api/saas/hives/{id}/status login
Returns provisioning status for a hosted hive: provisioning, running, or error with details.
DELETE /api/saas/hives/{id} owner
Delete a hosted hive. Removes K8s namespace, metadata, and user mappings. Owner or hub admin only.

Access Control

GET /api/saas/hives/{id}/access owner
List all users with access to a hosted hive and their roles.
POST /api/saas/hives/{id}/access owner
Grant access to a user. Body: {"username": "...", "role": "read"|"read-write"|"owner"}. Cannot have zero owners.
DELETE /api/saas/hives/{id}/access/{username} owner
Revoke access for a user. Cannot remove the last owner.
GET /api/saas/auth-check internal
Used by nginx auth-url annotation. Validates the user's cookie and checks access to the hive specified by ?hive={id}. Returns X-Hive-User and X-Hive-Role headers.

Hub Admin

GET /api/saas/admin/users admin
List all hub users with their profiles, quotas, hive mappings, and blocked status. Encrypted tokens are stripped.
PUT /api/saas/admin/users/{username} admin
Update a user's quota or blocked status. Body: {"saas_quota": N, "blocked": bool}. Quota -1 = unlimited, 0 = no access.

Spoke Integration

POST /api/heartbeat public
Hive spokes push their status here every eval cycle. Payload includes hive ID, org, repos, ACMM level, agents, governor state, contributors, leaderboard, health, owner, dashboard URL, snapshot URL.
POST /api/task-status public
Lightweight update from spokes every 30s with contributor/leaderboard data. Keeps current task info fresh between full heartbeats.
POST /api/saas/user-token login
Spoke requests a user's decrypted OAuth token for GitHub API calls. Body: {"hive_id", "username"}. Validates the user has access to the hive. Token is AES-256-GCM encrypted at rest.
